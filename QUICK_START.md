# 🚀 Quick Start Guide

## One-Click Setup (Recommended)

### **Windows Users:**
1. Double-click `setup.bat`
2. Follow the prompts
3. Application will start automatically!

### **macOS/Linux Users:**
```bash
# Make script executable (first time only)
chmod +x setup.sh

# Run setup script
./setup.sh
```

## Manual Setup (Alternative)

If the automatic setup doesn't work, follow the detailed guide in `SETUP_GUIDE.md`.

## What the Setup Script Does:

1. ✅ **Checks Python installation** (3.7+ required)
2. ✅ **Creates virtual environment** (.venv folder)
3. ✅ **Installs all dependencies** (Flask, scikit-learn, etc.)
4. ✅ **Sets up database** with 20+ companies
5. ✅ **Trains ML model** for recommendations
6. ✅ **Starts the application** automatically

## After Setup:

- **URL**: http://127.0.0.1:5000
- **Features**: AI recommendations, real job links, application tracking
- **Companies**: Google, Microsoft, Amazon, Zomato, Paytm, and more!
